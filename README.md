# Wizz Music

A program that allows you to control the Wizz lighting system in sync with music.

## Setup

### MacOS

1. Install "BlackHole" `brew install --cask blackhole-2ch`
2. Go to configure Audio MIDI, add multidevice with BlackHole and your audio output, name it "Loopback"
3. Set your device audio output to "Loopback"
4. `setup.sh`
5. `run.sh`
